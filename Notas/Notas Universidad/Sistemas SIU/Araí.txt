INSTALACION LDAP
----------------------------------------------------------------------------

1- git clone  https://hub.siu.edu.ar/siu-arai/arai-usuarios.git

2- cd /root/tmp/arai-usuarios/

3- cp idm/templates/ldap/3.1/ldif/cn\=\{* "/etc/ldap/slapd.d/cn=config/cn=schema"

4- chown openldap:openldap "/etc/ldap/slapd.d/cn=config/cn=schema" -R

5- systemctl restart slapd

6- netstat -npl | grep 389

7- ldapsearch -LLLQY EXTERNAL -H ldapi:/// -b cn=schema,cn=config "(objectClass=olcSchemaConfig)" dn

9- ll idm/templates/ldap/3.1/estructura-arai-usuarios.ldif

10- ldapadd -x -W -D "cn=admin,dc=universidad,dc=edu,dc=ar" -f idm/templates/ldap/3.1/estructura-arai-usuarios.ldif

11- vi idm/templates/ldap/3.1/estructura-arai-usuarios.ldif

12- ldapadd -x -W -D "cn=admin,dc=universidad,dc=edu,dc=ar" -f idm/templates/ldap/3.1/estructura-arai-usuarios.ldif

13- vi idm/templates/ldap/3.1/optional/memberOf.ldif

14- ldapmodify -Y EXTERNAL -H ldapi:/// -f idm/templates/ldap/3.1/optional/memberOf.ldif

15- ldapsearch -x -W -D "cn=admin,dc=universidad,dc=edu,dc=ar" -b "dc=universidad,dc=edu,dc=ar"

16- ldapsearch -LLLQY EXTERNAL -H ldapi:/// -b cn=schema,cn=config "(objectClass=olcSchemaConfig)" | grep memberOf

17- ldapmodify -Y EXTERNAL -H ldapi:/// -f idm/templates/ldap/3.1/optional/index.ldif

18- vi idm/templates/ldap/3.1/optional/index.ldif

19- ldapmodify -Y EXTERNAL -H ldapi:/// -f idm/templates/ldap/3.1/optional/index.ldif

20- systemctl stop slapd.service

21- systemctl start slapd.service

Limites - Limits y reservations en el archivo docs.yml

   limits:
       cpus: '1'
       memory: 4096M 
   reservations:
       cpus: '0.05'
       memory: 64M
	   
Los limits indican el máximo de recursos que va a usar el servicio, y los reservations indican el mínimo de recursos.
El servicio docs de arai documentos necesita extender su limits, asi como el bloque pdf y api en sudocu
